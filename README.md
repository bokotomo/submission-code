#提出用
1 ダウンロード/クローンする。(https://github.com/bokotomo/submission-code.git)

2 ディレクトリ内でrails sで実行

##使用言語
Ruby on Rails  
React.js  
SQLite  
HTML  
CSS

##実装した機能
・スキルを推薦機能  
・スキル/ユーザ一覧を表示  
・スキルのプラスボタンを押すと追加される機能  
・スキル詳細ページでのスキルを持ったユーザの一覧  
・スキル詳細ページでの自分のスキルへ追加ボタン
